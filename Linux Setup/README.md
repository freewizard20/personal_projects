# auto installing linux script for initial setup

## environment

> Ubuntu 16.04 with GTX 10 ~ GPU.

## Packages included

* python
* gcc
* java
* OpenGL
* Nvidia Driver 384
* CUDA 9.0
* CURNN 7.0
* tensorflow and various python libs
* various dynamic libraries

## How to use

`chmod +x setup.sh`

`sudo ./setup.sh`
